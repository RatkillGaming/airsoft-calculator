# Contributing to Airsoft Calculator

Thank you for your interest in contributing! This is a learning project, so contributions are welcome.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/airsoft-calculator.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Run tests: `npm test`
6. Commit your changes: `git commit -m 'Add some feature'`
7. Push to your branch: `git push origin feature/your-feature-name`
8. Open a Pull Request

## Testing

Please ensure all tests pass before submitting:

```bash
npm test
```

## Code Style

- Use TypeScript for type safety
- Follow existing code patterns
- Add comments for complex logic
- Keep components focused and reusable

## Adding New Tests

When adding new features, please add corresponding Playwright tests:

1. Add test cases to `tests/app.spec.ts`
2. Run tests to ensure they pass
3. Update test documentation if needed

## Questions?

Feel free to open an issue for questions or suggestions!
